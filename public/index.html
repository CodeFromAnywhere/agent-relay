How It Works

  1. Human interacts through various channels (WhatsApp, SMS, email, etc.)
  2. Agent Relay receives the message and routes it to the appropriate AI agent
  3. AI agent processes the request and generates a response
  4. Agent Relay sends the response back through the original channel
  5. Conversation continues seamlessly across multiple interactions
